# 🤖 BIT AI Agent Commander (v6.0.0)
## نظام القيادة المتكامل للوكلاء الأذكياء - الإصدار الخارق (BIT Hyper Edition)

> "BIT: Small Name, Infinite Power."
> "BIT: اسم صغير، قوة لا نهائية."

أهلاً بك في **BIT AI Agent Commander**، النظام الأكثر تطوراً وقوة في إدارة وتوجيه وكلاء الذكاء الاصطناعي المتعددين. هذا المشروع ليس مجرد أداة CLI عادية، بل هو **بنية تحتية متكاملة** مصممة لتمكين المطورين والمستخدمين المتقدمين من تسخير قوة نماذج اللغة الكبيرة (LLMs) في مهام معقدة، برمجية، وتحليلية بسرعة فائقة ودقة متناهية.

تم إعادة بناء هذا النظام بالكامل تحت هوية **BIT** ليكون أسرع، أخف، وأكثر ذكاءً، مع دمج تقنيات **BIT Hyper Scan** و **BIT-Search** لضمان أفضل النتائج في أقل وقت ممكن.

---

## 📖 الفلسفة وراء BIT

في عالم يتسارع فيه تطور الذكاء الاصطناعي، لم يعد الاعتماد على نموذج واحد كافياً. فلسفة **BIT** تقوم على مبدأ **"تضافر العقول الاصطناعية"**. بدلاً من سؤال نموذج واحد عن كل شيء، يقوم **BIT** بدور "القائد" (Commander) الذي يحلل طلبك، ثم يختار "الروبوت" (Robot) الأنسب لتنفيذه، أو حتى يوزع المهمة على عدة روبوتات تعمل معاً في تناغم تام.

هذا الإصدار (v6.0.0) يمثل نقلة نوعية نحو **الذكاء التراكمي والاستقلالية الكاملة**، حيث يتعلم العميل من كل تفاعل ويصبح أكثر قدرة على التكيف والتعلم الذاتي.

## ✨ الميزات الجديدة في الإصدار 6.0 (The Natural Network & Autonomous Agent Edition)

### 1. الشبكة العصبية الطبيعية (Natural Network)
**BIT** الآن يبني شبكة عصبية طبيعية خاصة به على جهاز المستخدم. هذه الشبكة تتكون من:
*   **الذاكرة الدائمة (Long-Term Memory):** كل محادثة، خطة، ونتيجة يتم حفظها وتحليلها بشكل دائم.
*   **المهارات المكتسبة (Learned Skills):** يتم استخلاص المهارات من المحادثات السابقة وتخزينها كخبرة للعميل.

**كيف تعمل؟**
قبل كل مهمة، يستعين "Agent Max" بهذه الشبكة لفهم سياق المستخدم بشكل عميق جداً، مما يعزز قدرته على:
*   **فهم دقيق لطلبات المستخدم:** حتى المهام المعقدة أو الغامضة يتم تحليلها بدقة "مرعبة".
*   **تعزيز ذكاء المزودين (LLMs):** تضيف هذه الشبكة طبقة ذكاء إضافية فوق نماذج الذكاء الاصطناعي الخارجية (مثل Gemini, OpenAI, Claude)، مما يجعل استجابات العميل أكثر ذكاءً وتخصيصاً.
*   **الاستعانة في المهام الصعبة:** تستخدم هذه الشبكة كـ "عقل إضافي" للعميل في المهام المعقدة، مما يمكنه من اتخاذ قرارات أكثر استنارة.

### 2. الاستقلالية في إدارة الأدوات (Autonomous Tooling)
العميل الآن قادر على تحديد الأدوات والاعتمادات (Dependencies) التي يحتاجها لتنفيذ مهمة معينة، وفي حال عدم توفرها، يقوم بتثبيتها تلقائياً داخل البيئة. هذا يضمن أن العميل يمكنه بناء المواقع، تشغيل الأكواد، أو استخدام أي أداة برمجية دون تدخل يدوي.

### 3. دعم شامل للمزودين (Universal Provider Support)
تم توسيع `BaseAgent.js` ليدعم التكامل السلس مع أي مزود ذكاء اصطناعي جديد، مع الحفاظ على استبعاد نموذج Grok من قائمة المزودين الافتراضية كما طلبت.

### 4. فحص وتصحيح الجسور (Bridge Health Check)
تم إضافة نظام فحص تلقائي لجسور التواصل (WhatsApp, Telegram, Email) للتأكد من عملها بشكل مستمر. في حال وجود أي مشكلة في الاتصال، يقوم النظام بمحاولة تصحيحها برمجياً لضمان استمرارية التواصل.

### 5. تدقيق شامل للكود (Global Code Audit)
تم إجراء مراجعة شاملة لجميع ملفات المشروع وتصحيح أي أخطاء منطقية أو نحوية لضمان عمل النظام بكفاءة واستقرار "على المسطرة".

## 🛠️ البنية التحتية والتقنيات المستخدمة (Deep Tech Stack)

تم بناء **BIT** باستخدام أحدث التقنيات لضمان أداء مستقر وقابل للتوسع:

### 1. المحرك الأساسي (Core Engine)
*   **Node.js (v18+):** تم اختيار Node.js لقدرته الفائقة على معالجة العمليات غير المتزامنة (Asynchronous Operations)، وهو أمر حيوي عند التعامل مع عدة وكلاء وطلبات API متزامنة.
*   **Commander.js:** الإطار العملي الذي يدير واجهة الأوامر (CLI)، مما يوفر تجربة مستخدم سلسة مع دعم كامل للوسائط (Arguments) والخيارات (Options).

### 2. واجهة المستخدم (User Interface)
*   **Blessed & Blessed-Contrib:** مكتبات قوية لبناء واجهات رسومية معقدة داخل الطرفية (Terminal UI). توفر لوحات معلومات (Dashboards)، رسوم بيانية، وقوائم تفاعلية.
*   **Chalk:** لإضافة الألوان والتنسيقات البصرية للأوامر، مما يسهل قراءة المخرجات والتقارير.
*   **Inquirer.js:** لإدارة الحوارات التفاعلية مع المستخدم أثناء الإعداد (Setup)، مما يضمن إدخال البيانات بشكل صحيح ومنظم.

### 3. أدوات الأتمتة والبحث (Automation & Intelligence)
*   **Puppeteer:** محرك تصفح (Headless Browser) يسمح للروبوتات بتصفح الإنترنت الحقيقي، تجاوز المواقع المعقدة (SPAs)، واستخراج البيانات بدقة لا تستطيع الـ APIs التقليدية الوصول إليها.
*   **Axios:** مكتبة قوية لإدارة طلبات HTTP، تستخدم للاتصال بمزودي الذكاء الاصطناعي المختلفين ومعالجة الردود بسرعة.
*   **BIT-Search Engine:** خوارزمية بحث مخصصة تدمج نتائج البحث التقليدية مع تحليل الوكلاء لتقديم إجابات دقيقة ومحدثة.

### 4. الأمان وإدارة البيانات (Security & Data)
*   **Dotenv:** لإدارة مفاتيح الـ API والمتغيرات الحساسة بعيداً عن الكود المصدري.
*   **JSON-based Config:** نظام إعدادات مرن يسمح بتخصيص سلوك كل روبوت ومزود بشكل مستقل.
*   **BIT Hyper Scan (ROIC):** أداة مدمجة لفحص الأكواد البرمجية وتحليل الثغرات الأمنية والأداء قبل الاعتماد النهائي.

## 🤖 جيش الروبوتات المتخصصة (The Elite Fleet)

يحتوي نظام **BIT** على أسطول يضم أكثر من 30 روبوت، كل منها يمتلك "بروتوكول" خاص وخبرة عميقة في مجاله:

### الروبوتات القيادية (Tier 1: Commanders)
| الروبوت | التخصص | القوة التشغيلية |
| :--- | :--- | :--- |
| **CodexBot** | هندسة البرمجيات الكاملة (Full-Stack) وتصميم المعماريات. | **3.0x** |
| **ArchitectBot** | تصميم الأنظمة المعقدة وأنماط التصميم (Design Patterns). | 2.0x |
| **AnalystBot** | تحليل البيانات الاستراتيجي واكتشاف الأنماط. | 2.0x |

### روبوتات التطوير والبرمجة (Tier 2: Developers)
*   **CodeGenerator:** متخصص في كتابة الأكواد بأكثر من 20 لغة برمجة.
*   **ReviewBot:** مراجع أكواد دقيق يكتشف الأخطاء المنطقية ويقترح تحسينات.
*   **TestingBot:** يقوم بكتابة اختبارات الوحدة (Unit Tests) واختبارات الضغط (Stress Testing).
*   **DevOpsBot:** خبير في خطوط الإنتاج (CI/CD)، Docker، و Kubernetes.
*   **SecurityBot:** متخصص في الأمن السيبراني واكتشاف الثغرات (Vulnerability Assessment).

### روبوتات البحث والمعرفة (Tier 3: Researchers)
*   **SearchBot:** محرك بحث بشري يحاكي تصفح الإنسان للوصول لأدق المعلومات.
*   **ResearcherBot:** يقوم بكتابة أبحاث أكاديمية وتقنية موثقة بالمصادر.
*   **VisionBot:** يمتلك "عيوناً" اصطناعية لتحليل الصور واستخراج النصوص منها (OCR).
*   **DataParser:** متخصص في تحويل البيانات غير المنظمة (Raw Text) إلى جداول وبيانات منظمة (JSON/CSV).

### الروبوتات المتخصصة (Tier 4: Specialists)
*   **LawBot:** تحليل العقود القانونية وتقديم استشارات قانونية أولية.
*   **FinanceBot:** تحليل الأسواق المالية، العملات الرقمية، واستراتيجيات التداول.
*   **HealthBot:** تقديم معلومات طبية مبنية على أبحاث (لأغراض إرشادية فقط).
*   **MathBot:** حل المعادلات الرياضية المعقدة والنمذجة الإحصائية.
*   **TranslatorBot:** ترجمة احترافية مع مراعاة السياق الثقافي والتقني.

## 🌐 المزودون والنماذج المدعومة (Multi-Provider Support)

نظام **BIT** لا يحصرك في خيار واحد، بل يفتح لك آفاقاً واسعة:

### 1. المزودون السحابيون (Cloud Providers)
*   **Google Gemini:** دعم كامل لنماذج Gemini 1.5 Pro و Flash (تتميز بنافذة سياق ضخمة).
*   **OpenAI:** دعم GPT-4o, GPT-4 Turbo, و GPT-4.1 Mini.
*   **Anthropic:** دعم عائلة Claude 3.5 (Sonnet & Opus) المعروفة بدقتها البرمجية.
*   **DeepSeek:** دعم نماذج DeepSeek-V3 و DeepSeek-Chat الاقتصادية والقوية.
*   **OpenRouter:** بوابة للوصول لمئات النماذج الأخرى مثل Llama 3 و Mistral.
*   **ملاحظة:** تم استبعاد نماذج xAI (Grok) من قائمة المزودين الافتراضية بناءً على طلب المستخدم.

### 2. النماذج المحلية (Local Models via Ollama)
يمكن لـ **BIT** العمل بالكامل على جهازك دون إنترنت باستخدام **Ollama**:
*   **Llama 3 (8B/70B):** نموذج ميتا القوي.
*   **Mistral/Mixtral:** النماذج الفرنسية المتفوقة.
*   **Qwen Coder:** نماذج علي بابا المتخصصة في البرمجة.
*   **DeepSeek-Coder:** أفضل النماذج المحلية للبرمجة حالياً.

## ⚙️ دليل التثبيت والتشغيل التفصيلي (Installation Guide)

### الخطوة 1: تجهيز البيئة
تأكد من تثبيت **Node.js** (الإصدار 18 أو أحدث) و **Git** على جهازك.

### الخطوة 2: استنساخ المشروع
```bash
git clone https://github.com/mohamed-dev-labs/bit.git
cd bit
```

### الخطوة 3: تثبيت التبعيات
```bash
npm install
```

### الخطوة 4: تهيئة النظام (Setup)
هذه أهم خطوة، قم بتشغيل الأمر التالي:
```bash
bit setup
```
سيسألك النظام عن:
1.  **المزود المفضل:** اختر من القائمة (مثلاً Google أو OpenAI).
2.  **مفتاح الـ API:** أدخل المفتاح الخاص بك (سيتم حفظه محلياً بشكل آمن).
3.  **النموذج الافتراضي:** سيقترح عليك النظام أفضل نموذج بناءً على اختيارك للمزود.
4.  **تثبيت ROIC:** سيقوم النظام تلقائياً باستنساخ أداة BIT Hyper Scan لتعزيز قدرات البرمجة.
5.  **تثبيت الأدوات الناقصة:** سيقوم العميل تلقائياً بتحديد وتثبيت أي أدوات برمجية إضافية يحتاجها بناءً على المهام المستقبلية.

## 🎮 طرق الاستخدام (Usage Modes)

### 1. الواجهة الرسومية (GUI Mode)
للحصول على تجربة احترافية، استخدم:
```bash
bit gui
```
ستفتح لك لوحة تحكم كاملة تظهر لك:
*   قائمة الروبوتات النشطة.
*   سجل العمليات (Logs).
*   نافذة الدردشة مع القائد.

### 2. وضع التنفيذ المباشر (Mission Mode)
لتنفيذ مهمة سريعة دون الدخول للواجهة:
```bash
bit run "قم ببناء تطبيق ويب بسيط لعرض حالة الطقس باستخدام React"
```

### 3. جسور التواصل (Bridges)
*   **WhatsApp:** `bit whatsapp` (يتطلب مسح كود QR).
*   **Telegram:** `bit telegram` (يتطلب توكن من BotFather).
*   **Email:** `bit email` (يتطلب إعداد Gmail App Password عبر `bit setup-email`).
